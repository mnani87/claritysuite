ClaritySuite Desktop (Hub · Forge · Toybox)

⬇ Download for macOS (v0.1)
Install steps (macOS, unsigned app):
  1. Download the ZIP and wait for the download to finish completely.
  2. Unzip it to get ClaritySuite.app. Double-click to open. On first launch you can also right-click the app, choose Open, and confirm.
  3. If macOS says the app is “damaged” or blocks it, run this once in Terminal:
    xattr -cr ~/Downloads/ClaritySuite.app
    Then open it again. (ClaritySuite is a local-only, unsigned app. macOS shows the "damaged" warning only because the app is not from the App Store or notarised.)
  4. Move ClaritySuite.app into your Applications folder.
⬇ Download for Linux (AppImage) (v0.1)
Install steps (Linux, AppImage):
  1. Download the AppImage file.
  2. Make it executable:
    chmod +x ClaritySuite-0.0.1.AppImage
  3. Run it by double-clicking, or from a terminal:
    ./ClaritySuite-0.0.1.AppImage
    The AppImage is self-contained and runs on most Linux distributions. No installation, root access, or dependencies are required.
