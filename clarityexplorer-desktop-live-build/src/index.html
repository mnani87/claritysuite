    Clarity Explorer

    A quiet, local-first workspace for research, reading, and thinking.

    Philosophy

    Clarity Explorer is a "Virtual Overlay" for your file system. Unlike traditional file managers, it lets you create "Projects" that group files from anywhere on your hard drive without moving or duplicating the originals. Your data never leaves this machine.

    Workflow

    1. Create & Add
    Create a project and click + to add files. You can mix PDFs, Docs, and Text files from different folders.
    2. Split View
    Read on the top, write notes on the bottom. Notes are saved to a local JSON file and linked by content hash, so they survive file renames.

    Limitations & Data Safety

    • Moved Files: If you move a file on your hard drive, Clarity will lose track of it. Use the "Locate" button in the preview pane to point to the new location. Your notes will be instantly migrated and restored.
    • PDF Search: Click inside the PDF and use Cmd+F to use the native finder.
